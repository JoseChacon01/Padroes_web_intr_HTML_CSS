O que é CSS?

Cascading Style Sheets é um mecanismo para adicionar estilo a um
documento web. O código CSS pode ser aplicado diretamente nas tags
"style" ou em vez de colocar a formatação dentro do documento, criar
um link para um arquivo CSS que contém os estilos.


Sintaxe

Na sintaxe de uma regra CSS, escreve-se o seletor e a seguir a propriedade
e o valor separados por dois pontos e colocados entre chaves { }.
